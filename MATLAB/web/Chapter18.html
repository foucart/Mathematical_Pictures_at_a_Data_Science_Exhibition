Adaptive group testing: binary splitting strategy

% generate a subset of [1:N] with size s
n = 10;
N = 2^n;
s = 5;
S = sort(randperm(N,s));
Here, the subsets are chosen adaptively, i.e., they depend on the outcomes of the previous tests. The observation and recovery procedures are thus intertwinned.
% recover each index from S one at a time, with n=log_2(N) tests each of them,
% resulting in m=s*log_2(N) adaptive tests in total
S_rec = [];
while ~isempty(setdiff(S,S_rec))
length_R = N; R_min = 1; R_max = N;
for i=1:n
length_R = length_R/2;
R_left = R_min:R_min+length_R-1;
R_right = R_min+length_R:R_max;
if isempty(intersect(S,setdiff(R_left,S_rec)))
R_min = min(R_right);
R_max = max(R_right);
else
R_min = min(R_left);
R_max = max(R_left);
end
end
new_idx = R_min;
S_rec = [S_rec, new_idx];
end
fprintf('The original and recovered supports agree:\n'), disp([S; S_rec])

Nondaptive group testing: recovery via a linear feasibility program

% generate a subset of [1:N] with size s
s = ceil(p/(d-1)) - 1;
S = sort(randperm(N,s));
x = zeros(N,1);
x(S) = ones(s,1);
% produce the vector of test outcomes
y = (A*x > 0);
% solve the linear feasibility program
I0 = find(y==0);
I1 = find(y==1);
cvx_quiet true
cvx_begin
variable z(N)
minimize 1
subject to
z >= 0;
A(I0,:)*z == 0;
A(I1,:)*z >= 1;
cvx_end
S_rec = find(z>1e-3)'; % note: find(z>0)' will work if the simplex algorihm is used
fprintf('The original and recovered supports agree:\n'), disp([S; S_rec])

Auxiliary function

Given integers , the function below produces the matrix with entries in whose columns indexed by polynomials f of degree contains the values modulo p.
function M = value_matrix(p,d)
M_1 = zeros(p,p);
for c=0:p-1
M_1(:,c+1) = c*ones(p,1);
end
if d==1
M = M_1;
else
M_aux = value_matrix(p,d-1);
pp = p^(d-1);
for c=0:p-1
M(:,c*pp+1:(c+1)*pp) = mod( c+diag(0:p-1)*M_aux , p);
end
end
end
