Chapter 16: Low-Rank Recovery from Linear Observations

% generate a rank-r matrix to be recovered from few linear observations
n = 60; % size of the square matrix
r = 3; % rank of the matrix
X = randn(n,r)*rand(r,n);
m = 1000; % number of observations

Recovery from generic observations via nuclear-norm minimization

Here, the observations made on the matrix take the form
where are gaussian matrices.
y_gen = zeros(m,1);
A = randn(n,n,m);
for i=1:m
y_gen(i) = trace(A(:,:,i)'*X);
end
tic;
cvx_begin quiet
variable X_gen(n,n)
variable P(n,n)
variable Q(n,n)
minimize trace(P)+trace(Q)
subject to
for i=1:m
trace(A(:,:,i)'*X_gen) == y_gen(i);
end
[P X_gen; X_gen' Q] == semidefinite(2*n);
cvx_end
t_gen = toc;
rel_error_gen = norm(X-X_gen,'fro')/norm(X,'fro');
fprintf(['Recovery performed in %.2f sec with relative Frobenius-error of %.2s',...
' using nuclear-norm minimization'], t_gen, rel_error_gen)
Recovery performed in 77.42 sec with relative Frobenius-error of 1.39e-08 using nuclear-norm minimization

Recovery from rank-one observations via nuclear-norm minimization

Here, the observations made on the matrix take the form
where are gaussian vectors.
y_rk1 = zeros(m,1);
a = randn(n,m);
b = randn(n,m);
for i=1:m
y_rk1(i) = b(:,i)'*X*a(:,i);

tic;

variable X_rk1(n,n)

subject to

end
[P X_rk1; X_rk1' Q] == semidefinite(2*n);

Recovery from rank-one observations via an iterative hard thresholding algorithm

The above semidefinite-programming-based recovery procedure is prohibitively slow (even if Mosek was selected as the solver).
In case of rank-one observations, a faster strategy consists in iterating the following scheme:
The operator H_r is the "hard thresholding operator" that returns the best rank-r approximant, i.e., the singular value deconposition truncated at level r.
The theoretical justification of this scheme's success can be found in [Foucart, Subramanian: Iterative hard thresholding for low-rank recovery from rank-one projections. Linear Algebra and its Applications, 572, 117--134, 2019].
nb_iter = 500;
X_iht = zeros(n,n);
tic;
for k=1:nb_iter
residual = y_rk1 - sum(b.*(X_iht*a))'; % this vector has entries y_i - < b^(i),X_k a^(i) >
M = b*diag(sign(residual))*a'; % this matrix is \sum_i residual_i b^(i) a^(i)^T
mu = norm(residual,1) / norm(M,'fro')^2;
[U,D,V] = svds(X_iht+mu*M,r);
X_iht = U*D*V';
end
t_iht = toc;
rel_error_iht = norm(X-X_iht,'fro')/norm(X,'fro');
fprintf(['Recovery performed in %.2f sec with relative Frobenius-error of %.2s',...
' using iterative hard thresholding'], t_iht, rel_error_iht)
Recovery performed in 1.60 sec with relative Frobenius-error of 1.25e-06 using iterative hard thresholding
